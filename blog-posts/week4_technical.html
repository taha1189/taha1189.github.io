Enumerable#group_by

The purpose of enumerable mixins is to allow the user the ability to sort throught data by traversing through it using different searching methods. The one in particular that we will be taking a look at is the enumberable group_by method. The group_by method sorts data into a hash. After traversing the data and searching for the specific sorting criteria described by the group_by method, the returned hash will consist of keys that are the evaluated result of searching the block and the values are the arrays of elements in the collection that are associated with the specific keys.

Example 1

(1..6).group_by {|i| i%3}
#=> {0=>[3, 6], 1=>[1, 4], 2=>[2, 5]}

Example 2

birds = ["Golden Eagle", "Gyrfalcon", "American Robin",
"Mountain BlueBird", "Mountain-Hawk Eagle"]
grouped_by_first_letter = birds.group_by { |s| s[0] }

# {"G"=>["Golden Eagle", "Gyrfalcon"], "A"=>["American Robin"],
# "M"=>["Mountain BlueBird", "Mountain-Hawk Eagle"]}
